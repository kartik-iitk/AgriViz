## How to Run?

1. Create a conda environment using `./environment.yml` using:

```bash
conda env create -f environment.yml
```

2. Download the necessary data for running the application from [here](https://drive.google.com/drive/folders/1g0kDE2Dn7YKDtF07N72_igaAxOzpxn_n?usp=sharing) and unzip it into `./data` folder.

3. Activate the environment using:

```bash
conda activate CS661
```

3. `cd` to the `./src` directory and run the code using:

```bash
python app.py
```

4. Open the browser and open `http://127.0.0.1:8050/`

Note: It may take some time to load the data and the initial pages of the dashboard. Please wait for the same.
